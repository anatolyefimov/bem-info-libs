← каталогБлок b-logo

﻿Блок использует внутри себя абстрактную иконку ##b-icon## и ссылку ##b-link##. Эти блоки миксируются в ##BEMHTML## шаблоне с соответствующими элементами блока ##b-logo##.

Обычно логотип представляет собой ссылку с изображением. ##BEMJSON## для этого варианта: %%hl js { block: 'b-logo', content: { elem: 'link', url: '/', title: 'logo', icon: { elem: 'icon', url: 'http://...', alt: 'logo' } } } %%

В некоторых случах, в ссылке, может находится не только иконка, но и текст, тогда ##BEMJSON## блока будет выглядеть следующим образом: %%hl js { block: 'b-logo', content: { elem: 'link', url: '/', content: [ { elem: 'icon', url: 'http://...', }, { elem: 'slogan', content: 'Trademark slogan' } ] } } %%

Если вам не нужна ссылка, то можно обойтись и просто изображением, ##BEMJSON## будет еще проще: %%hl js { block: 'b-logo', content: { elem: 'icon', url: '/', alt: 'logo' } } %%
