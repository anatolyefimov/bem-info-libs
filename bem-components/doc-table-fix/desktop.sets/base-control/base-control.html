base-control

Вспомогательный блок. Используется в работе большинства блоков библиотеки bem-components: attach, button, checkbox, input, link, menu, radio. Предназначен для реализации общей функциональности данных блоков, предоставляющей состояния focused и disabled.

Использует следующие методы:

Метод Описание
beforesetMod Сообщает блоку его действие до установки модификатора. Производит проверку активности блока: если блок неактивен (disabled), то он не может получить модификатор focused.
setMod Устанавливает блоку соответствующий модификатор (disabledили focused).
getName Возвращает имя контрола при его наличии. В обратном случае должен вернуть пустую строку.
getVal Возвращает значение контрола.
