menu

Блок menu используется для создания различных типов меню и селектов. Он предоставляет возможность изменять размер и внешний вид блоков меню, управлять поведением вложенных блоков – пунктов меню или селекта.

Варианты использования меню

Блок menu служит для создания различных типов меню:

Тип Описание Пример
Меню-переключатель Применяется для создания меню или селектов, позволяющих только одиночный выбор. 

    {
        block : 'menu',
        mods : { theme : 'simple', select : 'radio' },
        content : [
            {
                block : 'menu-item',
                val : 1,
                content : 'First item'
            },
            {
                block : 'menu-item',
                val : 2,
                content : 'Second item'
            }
        ]
    }
Селект с множественным выбором При щелчке мышью по пункту меню его состояние меняется на противоположное. Если пункт был активен – он деактивируется, и наоборот. 

    {
        block : 'menu',
        mods : { theme : 'simple', select : 'check' },
        content : [
            {
                block : 'menu-item',
                val : 1,
                content : 'First item'
            },
            {
                block : 'menu-item',
                val : 2,
                content : 'Second item'
            }
        ]
    }

Темы блока _theme

Блок представлен в следующих темах:

  • simple
  • normal

Без указания темы к блоку применяются значения по умолчанию (default), установленные браузером.

Размеры меню _size

Задает размер всем типам меню. Обязательный модификатор.

Реализован только в теме normal.

Доступно четыре размера реализации блока: s, m, l, xl.


Параметры/Размер s m l xl
Размер шрифта 13px 13px 15px 15px
Высота строки line-heigh 24px 24px 28px 32px
Левый отступ padding-left 30px 30px 34px 40px
Отступ padding 10px 13px 15px 20px

Одиночный или множественный выбор _select

Модификатор _select управляет возможностью множественного выбора пункта меню.

  • В значении radio при щелчке мышью будет производиться переключение активных пунктов меню. При повторном щелчке пункт, по которому он был произведен, останется активным;
  • В значении check пункт, по которому был произведен щелчек, изменит состояние на противоположное и будет менять состояние при повторных щелчках.

Реализован во всех темах блока.

Тип Пример
Одиночный выбор { block : 'menu', mods : { theme : 'normal', size : 'l', select : 'radio' }, content : [ { block : 'menu-item', val : 1, mods : { checked : true },
content : 'First item' }, { block : 'menu-item', val : 2, content : 'Second item' } ] }
Множественный выбор { block : 'menu', mods : { theme : 'normal', size : 'l', select : 'check' }, content : [ { block : 'menu-item', val : 1, content : 'First item' }, { block : 'menu-item', val : 2, mods : { checked : true }, content : 'Second item' } ] }

Состояния блока

В фокусе _focused

Модификатор focused в значении true автоматически выставляется блоку в момент, когда он находится в фокусе. Например, по нажатию клавиши Tab или при щелчке мышью.

Реализован во всех темах блока.

Элементы блока

Визуально представлен следующими элементами:

__group

Элемент __group служит для группировки пунктов меню. Пункты, которые нужно сгруппировать, помещаются в поле content элемента.

Например:

{
    block : 'menu',
    mods : { theme : 'normal', size : 'xl', select : 'radio', focused : true  },
    content : [
        {
            block : 'menu-item',
            val : 1,
            content : 'New'
        },
        {
            elem : 'group',
            content : [   
                {
                block : 'menu-item',
                mods : { checked : true },
                val : 2,
                content : 'Open'
                },
                {
                block : 'menu-item',
                val : 3,
                content : 'Open Recent'
                }
            ]
        },                                                                               
         {
            block : 'menu-item',
            val : 4,
            content : 'Open Not so Recent)'
        }
    ]
}

__group-title

Элемент позволяет задать заголовок для группы пунктов меню, создаваемой с помощью элемента __group.

Например:

    {
        block : 'menu',
        mods : { theme : 'normal', size : 'xl', select : 'radio', focused : true  },
        content : [
            {
                block : 'menu-item',
                val : 1,
                content : 'New'
            },                                  
            {
                elem : 'group',
                title : 'Cool title'                
                content : [   
                    {
                        block : 'menu-item',
                        mods : { checked : true },
                        val : 2,
                        content : 'Open'
                    },
                    {
                        block : 'menu-item',
                        val : 3,
                        content : 'Open Recent'
                    }]
            }
        ]
    }
