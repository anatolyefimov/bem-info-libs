image

Блок image используется для вставки картинок.

В HTML представлен тегом <img>.

Допустимые атрибуты блока

Допустимые атрибуты блока задаются в одноименных полях входного BEMJSON блока:

Атрибут Тип Описание
title String Задает имя картинки.
alt String Обязательный атрибут. Устанавливает альтернативный текст для изображений. Такой текст позволяет получить текстовую информацию о картинке при отключенной в браузере загрузке изображений.
Блок распознается программами чтения с экрана для альтернативного воспроизведения.
url String Адрес ссылки, по которой подгружается картинка.
width, height Number Значения, которые будут переданы в одноименные атрибуты картинки.
